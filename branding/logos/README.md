# AirTax Financial — Logo Assets

This folder stores all official AirTax Financial logo assets used across the website, marketing materials, and documents.

---

## Logo Usage Guidelines

All logo usage must remain consistent with the AirTax Financial brand guide.

---

## Primary Logo

The primary logo is the default AirTax Financial mark. Use it as the first choice whenever possible. Do not alter colors, proportions, or spacing. Do not add effects (shadows, outlines, gradients) unless specified in the brand guide.

---

## Dark Background Usage

On dark backgrounds (e.g., dark headers, footers, or dark imagery), use the approved light/inverted logo variant. Ensure sufficient contrast for readability. Do not place the primary (dark) logo on dark backgrounds.

---

## Light Background Usage

On light or white backgrounds, use the primary logo. Ensure the logo meets minimum contrast requirements. Avoid placing the light logo variant on light backgrounds.

---

## Minimum Spacing

Maintain clear space around the logo equal to the height of the "A" in AirTax (or equivalent reference) on all sides. Do not place text, images, or other elements within this clear space.

---

## File Types

| Format | Use |
|--------|-----|
| **SVG** | Web, scalable graphics; preferred for digital use |
| **PNG** | Web fallback, presentations; use with transparent background where applicable |
| **JPG** | Non-transparent backgrounds only |
| **PDF** | Print and document use |
