# Business Rules

## Service Pricing

**Standard Individual Tax Filing:** $249

---

## Aviator Network Discount

**Price:** $100 + 10,000 AeroCoins

---

## Purchase Code Rules

Each purchase generates:

- Unique code
- 4 digit verification PIN

---

## Code Lifecycle

Status values:

- AVAILABLE
- PURCHASED
- REDEEMED
- EXPIRED

---

## Code Expiration

Codes expire after 90 days.

---

## Unused Code Refund

If unused after 90 days:

Refund 22,000 AeroCoins to the user.

---

## Security Rules

- Codes are single-use
- Code + PIN required for redemption
- Once redeemed, the code becomes invalid.
