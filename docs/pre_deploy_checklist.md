# Pre-Deploy Checklist

Use this checklist before launching the AirTax Financial website or any major release.

---

## 1. Branding

- [ ] Brand guide in place and approved
- [ ] Logo assets ready (primary, secondary, favicon)
- [ ] Logo variants available (light/dark backgrounds, horizontal/stacked)
- [ ] Colors documented (primary, secondary, accent)
- [ ] Typography documented (headings, body, accents)
- [ ] Brand voice guidelines applied to all copy

---

## 2. Core Website Pages

- [ ] **Home** — Hero, value proposition, CTAs
- [ ] **Services** — Overview and links to all service pages
- [ ] **Contact** — Form, scheduling, contact info
- [ ] **About** — Company story, team, positioning
- [ ] **Pricing** — Transparent pricing for key services
- [ ] **FAQ** — Common questions answered

---

## 3. Trust Elements

- [ ] Privacy policy published and linked
- [ ] Terms of service published and linked
- [ ] Contact email visible and functional
- [ ] Business location or service area stated
- [ ] Professional disclaimer (non-legal, non-advice where applicable)
- [ ] Secure document handling communicated

---

## 4. Service Structure

- [ ] All product pages created and live
- [ ] Service links on Services page point to correct pages
- [ ] Product names consistent across website, products, and docs
- [ ] Services index (if used) matches current catalog
- [ ] Navigation includes all main services or clear groupings

---

## 5. Operations

- [ ] Client onboarding workflow documented
- [ ] Tax filing workflow documented
- [ ] Purchase code system documented
- [ ] Key processes accessible to team (internal links or shared location)

---

## 6. Launch Readiness

- [ ] No placeholder text (e.g., "Lorem ipsum," "To be defined")
- [ ] No broken links (internal or external)
- [ ] Mobile-friendly layout tested
- [ ] Clear call to action on every important page
- [ ] Forms tested (contact, scheduling, intake)
- [ ] Analytics and tracking configured (if applicable)
- [ ] HTTPS enabled
