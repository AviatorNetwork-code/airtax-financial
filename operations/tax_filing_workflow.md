# Tax Filing Workflow

## Overview

This document defines the standard workflow for individual tax preparation at AirTax Financial. It ensures consistency, quality, and compliance for every return—with particular attention to aviation-specific items.

---

## Principles

- **Precision** — Every return is reviewed for accuracy and optimal outcomes.
- **Documentation** — All client data and decisions are securely recorded.
- **Communication** — Clients are kept informed at each stage.

---

## Workflow Stages

### Stage 1: Intake & Engagement

| Step | Action | Owner |
|------|--------|-------|
| 1.1 | Receive client inquiry or referral | Operations |
| 1.2 | Schedule discovery call | Client Success |
| 1.3 | Conduct intake; collect prior-year return (if applicable) | Preparer |
| 1.4 | Issue engagement letter and fee agreement | Operations |
| 1.5 | Obtain signed engagement; collect deposit (if required) | Operations |
| 1.6 | Open client file in practice management system | Operations |

---

### Stage 2: Document Collection

| Step | Action | Owner |
|------|--------|-------|
| 2.1 | Send secure document request checklist | Client Success |
| 2.2 | Receive documents via secure portal or encrypted email | Client Success |
| 2.3 | Verify completeness (W-2s, 1099s, K-1s, etc.) | Preparer |
| 2.4 | Request aviation-specific items: | Preparer |
| | • Per diem statements / travel pay summaries | |
| | • Training receipts (BFR, type ratings, recurrent) | |
| | • Uniform / equipment receipts | |
| | • Union dues and professional association fees | |
| | • Aircraft ownership / rental documentation | |
| 2.5 | Follow up on missing items | Client Success |

---

### Stage 3: Preparation

| Step | Action | Owner |
|------|--------|-------|
| 3.1 | Assign preparer; load data into tax software | Preparer |
| 3.2 | Enter all income sources; reconcile to documents | Preparer |
| 3.3 | Process aviation-specific items per current guidance | Preparer |
| 3.4 | Apply deductions, credits, and elections | Preparer |
| 3.5 | Generate draft return; run diagnostics | Preparer |
| 3.6 | Internal review (second set of eyes) | Reviewer |
| 3.7 | Prepare client summary and cover memo | Preparer |
| 3.8 | Mark file as ready for client review | Preparer |

---

### Stage 4: Client Review & Approval

| Step | Action | Owner |
|------|--------|-------|
| 4.1 | Deliver draft return via secure portal | Client Success |
| 4.2 | Schedule review call (15–30 min) | Client Success |
| 4.3 | Walk client through summary; answer questions | Preparer |
| 4.4 | Document any changes requested | Preparer |
| 4.5 | Obtain verbal or written approval to file | Preparer |
| 4.6 | Collect e-file authorization (Form 8879) | Client Success |

---

### Stage 5: Filing & Follow-Up

| Step | Action | Owner |
|------|--------|-------|
| 5.1 | Transmit return to IRS and state agencies | Preparer |
| 5.2 | Confirm acceptance; store acknowledgment | Preparer |
| 5.3 | Send confirmation and copy of filed return to client | Client Success |
| 5.4 | Schedule post-filing review (tax planning) | Client Success |
| 5.5 | Archive file per retention policy | Operations |

---

## Aviation-Specific Checklist

Before marking a return as complete, verify:

- [ ] Per diem / travel allowances treated correctly (taxable vs. non-taxable)
- [ ] Flight training costs reviewed for deductibility
- [ ] Uniform and equipment deductions applied where appropriate
- [ ] Union dues and professional fees captured
- [ ] Multi-state allocation correct for pilots with varied domicile
- [ ] Aircraft-related income and expenses (rental, ownership) addressed
- [ ] Any special elections (e.g., per diem aggregation) applied correctly

## Timelines

| Stage | Target Duration |
|-------|-----------------|
| Intake to engagement | 3–5 business days |
| Document collection | Client-dependent; follow up at 7 and 14 days |
| Preparation | 5–10 business days (after documents complete) |
| Client review | 3–5 business days |
| Filing | Same-day after approval |

**Peak season:** Extend preparation timelines as needed; communicate clearly with clients.
